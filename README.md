# Русский штурман для BeamNG.drive (ралли)

Программа заменяет английскую озвучку штурмана в раллийном режиме BeamNG.drive на русскую.

Работает **внешне**, не вмешиваясь в игровые файлы (только минимальный патч для передачи данных).

## Как установить

1. Скачай последнюю версию из раздела [Releases](https://github.com/PetrosyanDobryakov/BeamNG.drive_Rally-RussianCoDriver/releases) (файл `RussianCoDriver_vX.X.X_full.zip`).
2. Распакуй архив в любое удобное место.
3. Запусти `RussianCoDriver.exe`.
4. При первом запуске программа попытается найти игру автоматически. Если не получится — укажи путь к `BeamNG.drive.exe` вручную.
5. Программа сделает резервную копию оригинального файла `audioManager.lua` и применит патч (нужно подтвердить права администратора).
6. Нажми кнопку **«Старт»**.
7. Запусти BeamNG.drive и начни раллийный заезд. В игре можно выключить громкость штурмана, чтобы не мешал оригинал.

## Как это работает

- При старте программа слушает UDP-порт 12347.
- В игру вносится минимальный патч, который отправляет структурированные фразы штурмана по UDP.
- Программа принимает фразы, разбирает их на компоненты (поворот, модификатор, дистанция) и проигрывает русские аудиофайлы из папки `audio`.

## Использование

- **Старт / Стоп** — запуск и остановка озвучки.
- **Громкость** — плавная регулировка.
- **Отладка** — разворачиваемая панель внизу окна.
- **Настройки (⚙)** — информация об авторах, путь к игре, проверка обновлений, ссылка для сообщений об ошибках.

## Обновление

Нажми в настройках «Проверить версию» — программа сама проверит наличие новой версии на GitHub и предложит скачать.

## Для разработчиков

Исходный код написан на Python 3 (используются только стандартные библиотеки + `requests` для проверки обновлений).

Запуск из исходников:

```bash
pip install requests
python BeamNGGUI.py
```
